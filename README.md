Script pyhon per convertire i file bin in chd se sono sprovvisti di file cue, converti bin scansiona i file bin di una cartella e ne genera file cue e chd, convertitutti bin invece fa lo stesso lavoro ma ne analizza tutte le sottocartelle se ad esempio ogni gioco dovesse avere una sua cartella. Utile sopratutto per buona parte dei giochi PS1 da mettere poi in retroconsole come batocera, recalbox, ecc. Utile quando si hanno molteplici file bin e non si è in possesso dei file cue, inoltre grazie al formato chd si ottiene un ottimo risparmio in termini si spazio di archiviazione.

ESSENZIALE: installare pacchetto mame-tools 
sudo apt install mame-tools
