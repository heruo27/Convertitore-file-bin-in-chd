Script pyhon per convertire i file bin in chd se sono sprovvisti di file cue, converti bin scansiona i file bin di una cartella e ne genera file cue e chd, convertitutti bin invece fa lo stesso lavoro ma ne analizza tutte le sottocartelle se ad esempio ogni gioco dovesse avere una sua cartella. Utile sopratutto per buona parte dei giochi PS1 da mettere poi in retroconsole come batocera, recalbox, ecc.
